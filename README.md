# nodejs-web-elm-reactor-hello-world

## Description
A POC elm program hello world.
Uses `reactor` command to directly 
render files.

## Tech stack
- npm
    - elm

## Docker stack
- node:latest

## To run
`sudo ./install.sh -u`
- [Availble](http://localhost/hello.elm)

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
